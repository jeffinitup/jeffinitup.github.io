I'm Jeff. 21 years old, male, based in the United States. I am a software developer. Software development to me is art; never really "seen" but absolutely felt in the tools we use and games we play day-to-day. To understand and create software has been an intense fixation of mine since I was a kid

Growing up I made little games using visual programming tools. I'd further my studies by getting into the practice of modifying existing games, which exposed me to a variety of design patterns and principles that I use to this day. I've remained curious, learning more about software structure, picking up new languages, and further improving the overall quality of my work.

My backlog of projects consists of games, game mods, and tools. You can learn more by visiting the site's "Projects" tab.

If you're curious, here are some of the coding languages I've worked with:

  • Java
  • C#
  • C++
  • GDScript
  • Python
  • LUA

I personally skew towards languages that are statically typed, and don't involve much manual memory management. With this in mind, Java and C# have been my favorites to work with! My projects tend to not require as much control as lower level languages give. I would eventually like to get more into C++ and try to pick up Rust, so I know how to use them when needed. (I suppose this is the natural progression of most programmers?)
